============
Installation
============


Stable release
--------------

To install LabOP Labware Ontology, run this command in your terminal:

.. code-block:: console

    $ pip install labop_labware_ontology

This is the preferred method to install LabOP Labware Ontology, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/
